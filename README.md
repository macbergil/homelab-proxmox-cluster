# Homelab Proxmox Cluster

Design and implementation of a multi-node Proxmox VE cluster with ZFS replication, backup strategy and infrastructure automation.

---

## 📐 Architecture

* 3-node Proxmox cluster
* ZFS-based storage with replication
* Dedicated backup node (Proxmox Backup Server)
* Internal networking for cluster communication

---

## 🧰 Technologies

* Proxmox VE
* ZFS
* Ansible
* (Planned) Terraform

---

## 🎯 Goals

* Reproducible infrastructure setup
* Automated configuration management
* Reliable backup and recovery strategy
* Scalable homelab architecture

---

## 📂 Repository Structure

* `ansible/` → configuration management
* `terraform/` → infrastructure provisioning (planned)
* `docs/` → architecture and setup documentation
* `scripts/` → helper scripts

---

## 🚧 Status

Work in progress – actively building and documenting.
